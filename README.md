# Costs

O **Costs** é uma aplicação para gerenciamento de projetos, permitindo criar, editar, remover projetos e adicionar/remover serviços em cada projeto. O objetivo principal é praticar conceitos fundamentais do React, como componentes, hooks, rotas, consumo de API e estilização com CSS Modules.

## Funcionalidades

- Cadastro de projetos com orçamento e categoria
- Listagem, edição e remoção de projetos
- Adição e remoção de serviços em projetos
- Validação de orçamento
- Feedback visual para ações do usuário

## Tecnologias Utilizadas

- React
- React Router DOM
- CSS Modules
- [json-server](https://github.com/typicode/json-server) para simular uma API REST
- React Icons

## Como Executar

1. Instale as dependências do projeto:
   ```sh
   npm install
   ```

2. Inicie o servidor fake (json-server):
   ```sh
   npm run server
   ```

3. Inicie a aplicação React:
   ```sh
   npm start
   ```

Acesse a aplicação em [http://localhost:3000](http://localhost:3000).
